Architecture:
    Insurance Policy PDF
            +
    Hospital Estimate PDF
            ↓
    PDF Text Extraction
            ↓
    Chunking
            ↓
    Hugging Face Embeddings
            ↓
    FAISS Vector Search
            ↓
    LLM Response
            ↓
    Coverage + Expected Payout + Missing Documents


Technology Stack:
    | Component   | Technology                             |
    | ----------- | -------------------------------------- |
    | Frontend    | Streamlit                              |
    | PDF Parsing | PyMuPDF                                |
    | Chunking    | LangChain                              |
    | Embeddings  | sentence-transformers/all-MiniLM-L6-v2 |
    | Vector DB   | FAISS                                  |
    | LLM         | Phi-3 Mini Instruct                    |
    | Deployment  | Hugging Face Spaces                    |
